# Taskbar Persistence

This method replaces taskbar **LNK** shortcuts to achieve persistence on a Windows system.

| 🇪🇸 Español | 🇺🇸 English |
|---|---|
| Con este método podemos lograr persistencia cuando el usuario interactúa con la barra de tareas. Básicamente, el método reemplaza los archivos **LNK** de la barra de tareas por el ejecutable que se desee.<br><br>Todo esto ocurre en la siguiente ruta:<br><br>`\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch\User Pinned\TaskBar`<br><br>Al reemplazar el archivo con el mismo nombre, Windows continúa utilizando la misma entrada del registro, pero apuntando a una dirección diferente.<br><br>Esto provoca que en la barra de tareas todo se vea igual (mismo icono y mismas propiedades), pero con una ruta de ejecución distinta. De esta forma, es posible ejecutar código malicioso para obtener persistencia haciéndolo pasar por otra aplicación del sistema operativo. | With this method, persistence can be achieved when the user interacts with the taskbar. The technique replaces the taskbar **LNK** files with any executable of your choice.<br><br>This occurs in the following path:<br><br>`\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch\User Pinned\TaskBar`<br><br>When the file is replaced using the same name, Windows continues to use the same registry entry, but it now points to a different location.<br><br>As a result, the taskbar appears unchanged (same icon and properties), but the execution path is different. This allows malicious code to execute and maintain persistence while appearing to be another operating system application. |
